Solve systems of equations step-by-step using elementary row operations — supports exponential coefficients, decimal values, and mixed equations.

✦
Three Elementary Row Operations
I
↕
Row Swap
Exchange two rows of the augmented matrix. Changes row order without affecting the solution set.
Rᵢ ↔ Rⱼ
II
×
Row Scaling
Multiply every element of a row by a non-zero scalar constant. Preserves all solutions.
Rᵢ → k·Rᵢ  (k ≠ 0)
III
⊕
Row Replacement
Replace a row with the sum of itself and a scalar multiple of another row — core of elimination.
Rᵢ → Rᵢ + k·Rⱼ
